## 一、5 大特征

### 1. 人口规模巨大的现代化
- **14 亿人口**(全球 18%)
- 美国 3.3 亿 / 欧盟 4.5 亿 / 日韩 1.8 亿
- 14 亿 = **"现代化史" 累计工业化人口 12 亿** ——
- **"中国式现代化" = 14 亿人的现代化**
- **不是"西方现代化的中国版"**

### 2. 全体人民共同富裕的现代化
- 不是"少数人富"
- 不是"平均主义"
- 是"**中等收入群体为主体**(橄榄型)
- 详见 97-共同富裕

### 3. 物质文明与精神文明相协调的现代化
- 不是"GDP 唯一"
- 强调"**两个文明"** 并抓
- 详见 95-脱贫 / 98-小康

### 4. 人与自然和谐共生的现代化
- 不是"先污染后治理"
- 强调"**绿水青山就是金山银山**"
- 2012-2024 碳排放强度降 50%
- 2030 碳达峰 / 2060 碳中和

### 5. 走和平发展道路的现代化
- 不是"国强必霸"
- 强调"**构建人类命运共同体**"
- 一带一路 / 全球发展倡议 / 全球安全倡议
- 不是"殖民 + 战争"剧本

## 二、与"西方现代化" 的 4 大差异

| 维度 | 西方现代化 | 中国式现代化 |
|---|---|---|
| **路径** | 殖民 + 奴隶制 + 战争 | 独立自主 + 改革开放 |
| **人口** | 3-4 亿(美/欧) | 14 亿(占全球 18%) |
| **分配** | 贫富差距大(基尼 0.4-0.5) | 共同富裕(目标基尼 0.4) |
| **生态** | 先污染后治理 | 绿水青山 + 双碳 |
| **文化** | 个人主义 | 集体主义 + 家庭 |
| **制度** | 多党制 | 党的领导 + 人民当家作主 + 依法治国 |
| **对外** | 霸权 / 战争 | 和平 / 共商共建共享 |

**核心**:**"中国式现代化"** = **"西方现代化" 的替代选项** ——
**不是"差的",是"不同的"**。

- 国有 + 民营 + 外资"三主体"

### 3. "全过程人民民主" (2019-)
- 党的领导 + 人民当家作主 + 依法治国
- "**选举 + 协商 + 决策 + 监督**" 全过程
- 不是"一选了之"
