## 一、开篇钩子

公元前 445 年,魏文侯即位。

他是**三家分晋**(前 403 年)后**魏国第一代"开国之君"** — 但魏国**只是"中等强国"**,西有秦,东有齐,南有楚,**被四面包围**。

魏文侯**问李悝**:**"为国何如?"** — "怎么治国?"

李悝对:**"**食有劳而禄有功,使有能而赏必行,罚必当。**"**

翻译:**"吃饭要给有功劳的人,做官要给有能力的人,赏赐要真的执行,惩罚要恰当。**"

这是**"战国变法"的第一声号角**。

李悝**4 项改革** + 吴起**军事改革** = **战国变法"前传"** — **为商鞅变法铺平道路**。

**李悝的《法经》** — 中国第一部**完整法典**(前 407):
- 6 篇:**盗法 / 贼法 / 网法 / 捕法 / 杂法 / 具法**
- 早于古罗马**《十二铜表法》**(前 450) 约 40 年
- 后被**商鞅"秦律"**继承
- 秦律再被**汉律 / 唐律**继承
- **2000 年中国法律的"源头"**

**李悝+吴起 = 战国"法治"的奠基者**。

## 二、背景速览

**魏国的"先天条件"**:
- **三家分晋**(前 403)后,魏国得到**晋国核心领土**(今山西西南 + 河南北部)
- 魏国**地理位置**好 — **居"天下之中"**(四方都通)
- 魏国**资源**丰富 — **盐、铁、煤**产量高
- 但**无"霸主地位"** — 晋国曾是"霸主",魏国只是"晋国一部分"

**魏文侯的"任务"**:
- **建立"魏国"** — 从"晋国的一部分"变成"独立强国"
- **任用贤才** — 招纳天下英才
- **变法图强** — 让魏国"富强"

**魏文侯的"成就"**:
- **招纳 4 位"师友"**:**卜式(师)、师文(友)、西门豹(邺令)、李悝(丞相)**
- **任用"魏武卒"** — 让吴起训练"精锐之师"
- **"称霸中原"** — 在战国初期是"中原最强"

**李悝的"任务"**:
- **改革内政** — 让魏国"富"
- **变法立法** — 让魏国"有制度"
- **培养接班人** — 培养"商鞅"

**吴起的"任务"**:
- **改革军事** — 让魏国"强"
- **训练"魏武卒"** — 中国最早的"精锐之师"
- **楚国变法** — 后期在楚国"再次变法"(失败)

## 三、故事深读

**"食有劳而禄有功":最早的"绩效管理"**

李悝**"食有劳而禄有功"** = **"绩效管理"** 的**最早理念**。

**之前的"分封制"**:
- **世袭贵族** — 父辈的"官位"传给孩子
- **无功受禄** — 很多人"什么都不做,只是贵族"
- **结果**:国家**养"闲人"**

**李悝的"食有劳而禄有功"**:
- **吃饭**(食)= "有功劳的人" 才能吃得好
- **做官**(禄)= "有功劳的人" 才能当官
- **结果**:**人人"求功"**,国家**有效率**

**李悝的"绩效管理"是"商鞅"和"现代企业"的"祖宗"**:
- **商鞅**继承李悝"食有劳而禄有功",变成**"军功爵制"**(有军功的人获得爵位)
- **现代企业** = "绩效管理" + "OKR" + "KPI"
- **"吃 + 禄 + 赏 + 罚"** = 2000 年来"绩效管理"的**核心**

**"平籴法":最早的"宏观调控"**

李悝的**"平籴法"** = **政府"控制粮价"** = **最早的"宏观调控"**。

**"平籴法"的操作**:
- **丰年**:政府**高价收购**粮食(让农民有收入)
- **灾年**:政府**低价抛售**粮食(让百姓买得起)
- **目标**:**粮价稳定** — 不让"丰年贱卖"和"灾年饿死"

**"平籴法"的本质** = **"国家储备"** + **"反周期"**:
- **国家储备** — 政府"有粮"
- **反周期** — 丰年买,灾年卖 — **不让粮价"暴涨暴跌"**

**"平籴法"在 2000 年后的现代应用**:
- **国家粮食储备** — 中国"中储粮" = 现代版"平籴法"
- **国家石油储备** — 各国"战略石油储备" = 油版"平籴法"
- **美联储 QE / QT** — 美联储"印钱买债"+"缩表卖债" = 美元版"平籴法"

**"平籴法"是"现代央行的祖宗"** — **2000 年前,中国就有"国家宏观调控"**。

**"武卒":最早的"精锐之师"**

李悝 + 吴起的**"武卒制"** = **中国最早的"精锐之师"**。

**"武卒"的选拔标准**:
- 全副武装(铠甲 + 头盔 + 戈 + 剑 + 弓 + 50 支箭 + 3 天口粮)
- **100 步内拉开 12 石弩**(约 720 斤)
- 携带装备**100 里(约 50 公里)半天到达**
- 通过选拔 = **"武卒"**

**"武卒"的奖励**:
- **赐田宅** — 给房子和田
- **免徭役** — 不需要服国家劳役
- **提升社会地位** — 周围人"尊重"

**"武卒"的退役**:
- 老了退役 = **保留"田宅"和"免徭役"**
- 退役的"武卒" = 魏国的"中产阶级"
- "武卒"退役后,他们**保卫国家** — 因为他们"有田宅"

**"武卒"是"商鞅军功爵制"的"前传"**:
- **商鞅"秦军功爵制"** = 升级版"武卒"
- **现代"美军"** = 升级版"武卒"(美军退伍有优厚待遇)
- **"武卒"是"现代职业军人"的最早原型**

**《法经》:中国第一部完整法典**

**李悝《法经》**(前 407 年)= **中国第一部完整法典**。

**《法经》6 篇**:
- **《盗法》** — 盗窃罪
- **《贼法》** — 杀人罪、伤害罪
- **《网法》** — 诈骗、诬告
- **《捕法》** — 逮捕、审讯
- **《杂法》** — 杂项(婚姻、债务、财产)
- **《具法》** — 量刑(根据"情节"决定"刑罚")

**《法经》vs 古罗马《十二铜表法》**:
- **《法经》** = **前 407 年**(早)
- **《十二铜表法》** = **前 450 年**(晚)
- 但《十二铜表法》更**系统化**,后影响**古罗马法**
- 《法经》**失传**(秦焚书坑儒),但被**汉律 / 唐律**继承

**《法经》vs《商君书》(商鞅)**:
- **《法经》** = **"刑法"** — 关注"什么行为犯什么罪"
- **《商君书》** = **"政论"** — 关注"为什么变法 + 怎么变法"
- **两者结合** = **"商鞅变法"** — 既有"法"也有"论"

**李悝 → 商鞅 → 韩非**:
- **李悝**(前 455-前 395) — **"法"**
- **商鞅**(前 390-前 338) — **"法 + 术"**
- **韩非**(前 280-前 233) — **"法 + 术 + 势"**
- **法家"三代"** = **"法"的演进**

**吴起"杀妻求将":变法者的"极端"**

吴起**"杀妻求将"** = **中国"极端改革者"的最早代表**。

**"杀妻"的故事**:
- 吴起是**卫国左氏(今山东曹县)人**
- 娶了**齐国女子**为妻
- **鲁君**想用吴起为将,但**"齐妻"**让鲁君不放心
- 吴起**杀了妻子** — 表明"不亲齐" — 鲁君才**用他为将**

**"杀妻"的政治意义**:
- 吴起**表明"不偏袒齐国"** — 让鲁君放心
- **"杀妻"= "对君王的绝对忠诚"** — 这是**"极端"**但**有效**
- 2000 年来,**"杀妻求将"**成为"极端忠诚"的代名词

**吴起**和**商鞅** = **"法家极端改革者"的两大代表**:
- **吴起**:"杀妻求将"(极端"私"层面的忠诚)
- **商鞅**:"立木为信"(极端"公"层面的法治)

**吴起"杀妻"vs 商鞅"立木"**:
- **吴起"杀妻"** = **"我忠于国君"** — **私**
- **商鞅"立木"** = **"法律面前人人平等"** — **公**
- **两个方向**:**"私"忠诚**和**"公"法治**

**吴起"变法" vs 商鞅"变法"**

**吴起在楚国变法**(前 386 - 前 381):
- **楚悼王**用吴起为相
- **变法修律** — 制定楚国法律
- **废疏远者** — 废除"远支贵族" — 削弱贵族
- **养战斗之士** — 奖励"战斗英雄" — 强军

**吴起"楚之变法" 5 年**(前 386-前 381)后:
- **楚悼王死**(前 381)
- **楚国贵族反扑** — **射杀吴起** — **变法失败**
- **楚国变法** = **"5 年"** = **"失败"**

**商鞅"秦之变法" 20 年**(前 356-前 338):
- **秦孝公**用商鞅为相
- **变法修律** — 制定秦律
- **废世袭** — 废除"世袭贵族" — 削弱贵族
- **军功爵制** — 奖励"军功" — 强军
- **县制 + 连坐法** — 郡县制 + 什伍连坐

**商鞅"秦之变法" 20 年** + **秦孝公死后商鞅被车裂**(前 338) — **变法人死,法律仍在**
- 秦**没有**"贵族反扑" — 因为商鞅"已经杀掉大部分贵族"
- 秦**继续**用"商鞅之法" → **秦统一六国**(前 221)

**吴起 vs 商鞅**:
- **吴起**:**温和变法** — **贵族反扑** — **变法失败**
- **商鞅**:**极端变法** — **杀掉贵族** — **变法成功**

**"温和 vs 极端"** 的差别:
- **温和变法** = 触犯"既得利益者"但**不"杀掉"**他们 — 他们**反扑**
- **极端变法** = 触犯"既得利益者"**而且"杀掉"**他们 — 他们**无法反扑**

**真正长久的改革** = **不"温和",也不"极端"** — **"改革 + 制度化"**:
- **改革** — 触动既得利益
- **制度化** — 让改革"不可逆"
- **商鞅成功** = **"改革"+"杀掉反对者"** — 制度化"不可逆"

## 四、现代映射

**1. "食有劳而禄有功" → 公司"绩效管理"**

"食有劳而禄有功" = **公司"绩效管理"**。

现代公司"绩效管理"案例:
- **华为"奋斗者协议"** — 奋斗者获得更多股权
- **Netflix "Top Grader"** — 顶级员工获得顶级奖励
- **字节"期权激励"** — 贡献者获得期权
- **小米"股权激励"** — 早期员工获得股权

**"绩效管理"的核心**:
- **"给有功劳的人"** — **不"平均主义"**
- **"有功劳"** = **可量化的"贡献"**(不是"工龄"或"关系")

**2. "平籴法" → 公司的"价格控制"**

"平籴法" = **公司"价格控制"**。

现代公司"价格控制"案例:
- **苹果 iPhone "定价策略"** — 丰年(新机发布)高价,灾年(老机清仓)低价
- **亚马逊"动态定价"** — 不同时段不同价格(机票、酒店)
- **特斯拉"价格调整"** — 2018 年多次调价(扩产能)
- **美团"补贴策略"** — 丰年(竞争激烈)补贴,灾年(垄断)不补贴

**"价格控制"的核心**:
- **反周期** — 不"顺周期"加价
- **"消费者"** — 长期**让"消费者"赢**
- **"竞争对手"** — 让"竞争对手"输

**3. "武卒" → 公司"关键人才激励"**

"武卒" = **公司"关键人才"**。

现代公司"关键人才激励"案例:
- **华为"天才少年"** — 2019 年起,招"天才少年",年薪 200 万+
- **字节"TopBuzz 人才"** — 关键岗位 + 股权激励
- **腾讯"长青计划"** — 关键员工 + 长期股权
- **小米"工程师"** — 关键工程师 + 期权 + 房补

**"关键人才激励"的核心**:
- **"找对人"** — 选拔标准严格
- **"留住人"** — 长期激励
- **"用对人"** — 让他"愿意打仗"

**4. "吴起杀妻" → 公司"创始人"vs"配偶"**

"吴起杀妻" = **公司"创始人"vs"配偶/家庭"** 的极端案例。

现代公司"创始人"vs"家庭":
- **乔布斯 1985 年被赶出苹果** — **妻子 Laurene Powell 仍支持** — **2007 年回归**
- **贝索斯 2019 年离婚** — **前妻 MacKenzie 拿到 25% Amazon 股份** — **"好聚好散"**
- **马斯克 多次离婚** — **多次** — **"工作狂"代价**
- **扎克伯格 一直结婚** — **妻子 Priscilla Chan 是重要支柱**

**"创始人"vs"家庭"** 的智慧:
- **不"杀妻"** — 真正的"伟人"靠"家庭支持"
- **不"工作狂"** — "工作狂"长期会"burnout"
- **"平衡"** — 创始人要"平衡"工作和家庭

## 五、一句话收尾

李悝+吴起教给我们的核心,**不是"变法",而是"变法的代价"** — **真正长久的"变法",既要"触动既得利益者",又要"防止反扑"**。**吴起"温和变法"被反扑,商鞅"极端变法"成功** — **历史告诉我们:"变法"不是"理想",是"权力的较量"**。

## 六、互文段落

跟 **2-商鞅变法**(第 2 篇)互文 — **商鞅是李悝的学生**。**李悝《法经》→ 商鞅"秦律"** = **"法家"两代传**。**商鞅"军功爵制" = 李悝"食有劳而禄有功"的升级版**。

跟 **7-长平之战**(第 7 篇)互文 — **"魏武卒"(吴起)vs "秦锐士"(商鞅)** = **战国最强的两支军队**。**魏武卒 5 万 vs 秦锐士 50 万** = **赵括 40 万大军的败局**。**变法的"温和 vs 极端" = 军力的"中等 vs 顶级"**。
